Two Lines of Development

The OpenSLP project developers have chosen to maintain two code branches. The Stable branch and the Development branch.

The current stable version is 2.0.0. Version 2.0 has been in development for many years and supports significant new functionality, including full IPv6 support, full asynchronous client API support, and several useful transparent extensions to the SLP standard.

The current development version is 2.1.0. Version 2.1 is slated to include features like mdns integration, and a few other less transparent extensions to the SLP standard.

The OpenSLP project also provides a pure Java implementation of the SLP API. The current version is the first version and has been stable since 2003.
