Release Notes for 1.2.0 (stable)

The latest stable version of OpenSLP is 1.2.0.  OpenSLP-1.3.0 contains a few minor bug fixes that some OpenSLP users will be interested in.  See ChangeLog in tarball for details.

Those that are investigating OpenSLP as part of a development or porting project should look at the 1.3.x development releases. This is where the development action will be. Those that are deploying OpenSLP in production systems should continue to look at OpenSLP 1.2.0 until the 1.3.x code becomes stable enough for production use.

This release includes all of the code updates and feature changes up to 1.1.5. The 1.2 branch will concentrate on bug fixes and minor enhancements, while the 1.3 branch is where major enhancements and changes will be made.

Release Notes for 1.3.0 (development)

IPv6 support has been added to OpenSLP. It is generally transparent to the API user, but it would be a good idea to look at the conf file additions to acquaint yourself with the new settings available. Both IPv4 and IPv6 are used by default, but this is easily changed in the conf file.
