Ports

OpenSLP is known to compile and run on the following platforms:
  • Linux
  • Win32
  • SCO UnixWare 7
  • FreeBSD
  • Solaris
  • Tru64
  • Mac OS X
  • Darwin
Please contact the OpenSLP maintainer if you know of an additional platform that will build and run OpenSLP or if you would like to see OpenSLP ported to a platform where it does not currently work.
